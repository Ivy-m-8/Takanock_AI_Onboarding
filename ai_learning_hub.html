Claude

Introduction to Claude Chat

Claude Chat is your AI assistant for writing, research, analysis, brainstorming, and more. This module covers the basics of getting great results.

What is Claude?

Claude is an AI assistant made by Anthropic. You can have natural conversations with it, ask questions, get feedback on documents, draft emails, summarize content, and much more.

Claude is available via claude.ai in your browser, the Claude mobile app, and through integrations your company may have set up.

How to write a good prompt

The quality of Claude's response depends a lot on how you frame your request. Think of it like briefing a smart colleague.

Be specific about what you want. Instead of "write an email," try "write a short follow-up email to a client after a sales demo."
Give context. Who is the audience? What's the goal? What format do you need?
Include examples if you have them — "write it in the style of this example: ..."
Iterate. If the first response isn't quite right, ask Claude to adjust it.
Pro tip: Claude remembers everything within a conversation. You can follow up with "make it shorter" or "make it more formal" without repeating your original request.

Example use cases

Summarize
"Summarize this 10-page report into 5 bullet points for an executive audience."
Draft
"Draft a project status update email for the Monday standup."
Analyze
"What are the main themes in this customer feedback survey?"
Brainstorm
"Give me 10 ideas for our Q3 team offsite."

Things to keep in mind

Claude can make mistakes. Always review important outputs before sending or publishing. Never share sensitive passwords, personal data, or confidential information beyond what your company policy allows.

Cowork

Cowork is a desktop app from Anthropic that lets Claude help you with files, tasks, and workflows directly on your computer — without switching back and forth to a browser.

What Cowork does

Cowork brings Claude to your desktop so it can see what you're working on and help you get things done. It can read files, draft content, organize information, and automate repetitive tasks — all from a persistent side panel.

Key difference from Claude Chat: Cowork works alongside your desktop applications, making it ideal for file-heavy or multi-app tasks.

Common use cases

Document work

Summarize, rewrite, or extract info from local files

Task management

Organize to-do lists and manage project files

Email drafting

Draft responses based on open emails or attachments

Repetitive tasks

Automate file renaming, formatting, and data cleanup

Getting started

Download and install Cowork from claude.ai/download (or your company's software portal).
Sign in with your Anthropic account (or your company SSO if configured).
Open the side panel and describe what you need help with. Drag and drop files into the chat to have Claude work with them.

Claude Code

Claude Code is a command-line tool that brings AI-assisted coding directly into your terminal. It's built for developers who want help writing, reviewing, and debugging code.

What it does

Claude Code runs in your terminal and has direct access to your codebase. It can read files, make edits, run commands, and help you work through complex engineering tasks step by step.

Best for: Developers who prefer a terminal workflow and want Claude to work directly in their project — not just give advice.

Example tasks

Write code
"Add a function to this file that validates email addresses."
Debug
"I'm getting a TypeError on line 42. What's wrong and how do I fix it?"
Refactor
"Refactor this class to follow the single responsibility principle."
Review
"Review this pull request and flag any security concerns."

Getting started

Install via npm: npm install -g @anthropic-ai/claude-code
Navigate to your project directory in the terminal.
Run claude to start a session. Claude will read your project structure automatically.

Projects

Projects in Claude let you organize conversations, share context across chats, and collaborate with teammates — all in one place.

What is a Project?

A Project is a dedicated workspace in Claude that keeps related conversations, uploaded files, and custom instructions together. Instead of starting fresh every time, Claude remembers the context of your project.

Think of it like: A shared folder + a briefed assistant. You upload your docs once, set your instructions, and every conversation in that Project benefits from that context.

Key features

Upload files

Add PDFs, docs, spreadsheets for Claude to reference

Custom instructions

Tell Claude your role, tone preferences, and rules

Share with team

Invite colleagues to collaborate in the same Project

Persistent context

Claude remembers project details across sessions

Creating a Project

In Claude Chat, click "New Project" in the left sidebar.
Give your project a name and write a brief description of what it's for.
Upload relevant files and add custom instructions (e.g., "Always respond in formal English").
Invite teammates via their email address. Start chatting!

Customize: Skills & Connectors

Claude can be connected to your apps and extended with skills to make it more powerful for your specific needs.

Key terms

API

Application Programming Interface — rules that let software talk to other software. When Claude connects to Slack, Airtable, or any service, it uses that service's API.

MCP

Model Context Protocol — an open standard by Anthropic that gives AI a universal "plug" to connect to tools like Notion, GitHub, and Airtable without custom code per tool.

SDK

Software Development Kit — a pre-built library that makes calling an API easier. Anthropic has SDKs for Python and JavaScript so developers can call Claude in one line of code.

JSON

JavaScript Object Notation — a lightweight, structured format for exchanging data between systems. Most APIs send and receive data as JSON.

Webhook

A URL that "lights up" when triggered by an event (like a form submission). AI agents are often started by a webhook — when it's hit, the workflow begins.

Connectors

Connectors let Claude access and act in your other tools — like reading Slack messages, searching your Google Drive, or creating Notion pages. Once connected, you can ask Claude to work across apps naturally.

Example
"Find all emails from last week about the product launch and summarize the key decisions."

Available connectors include:

Slack
Google Drive
Notion
Google Calendar
Gmail
Airtable
Asana
And more...

How to add a connector

Open Claude Chat and go to Settings → Connectors.
Browse available connectors and click "Connect" next to the app you want.
Authorize access in the popup (you'll sign in to the third-party app).
Start using it — just mention the app in your prompt and Claude will use it.

Skills & preferences

You can customize how Claude behaves by setting preferences in your profile. For example, tell it your job title, preferred writing style, or how detailed you want responses to be.

User preferences (under your profile) let you set standing instructions like: "I'm a product manager. Always be concise. Use bullet points for lists."

Building AI Agents

AI agents go beyond answering questions — they can take actions, use tools, and complete multi-step tasks on your behalf.

What is an AI agent?

An AI agent is an AI system that can plan and execute a series of steps to complete a goal. Unlike a simple chatbot that responds to one question at a time, an agent can:

Receive a high-level goal (e.g., "research competitors and summarize findings")
Break it into subtasks and decide what order to do them in
Use tools like web search, file access, or APIs to gather information
Synthesize results and deliver a final output

When to use an agent

Use a chatbot when...Use an agent when...
You need a single answer or piece of contentThe task has multiple steps or decisions
You'll review and act on the output yourselfYou want the AI to take actions automatically
The task is well-definedThe task requires gathering info from multiple sources

Key concepts

Tools

Functions the agent can call: search, read files, send messages

Planning

How the agent decides what steps to take next

Loops

The agent can retry or self-correct when something fails

Human review

For critical steps, agents can pause and ask for approval

Claude's agents: Claude supports agentic tasks natively — try asking it to "research X and write a report" and watch it plan and execute multiple steps.

Building your own agents

You can build custom agents using tools like the Anthropic API, Claude Projects with connectors, or workflow automation platforms like n8n or Zapier. See the Automating Workflows module for more.

Automating Workflows with AI

AI can be wired into your existing tools and processes to reduce manual work, eliminate repetitive tasks, and respond faster to events.

What is workflow automation?

Workflow automation connects apps and triggers actions automatically. Adding AI to the mix means you can handle tasks that previously required human judgment — like drafting a response, categorizing a ticket, or summarizing a document.

Example: When a new support ticket arrives → AI classifies it → assigns it to the right team → drafts a suggested reply → sends it for human review.

Popular AI automation tools

n8n

Visual node-based workflow automation — the "plumbing" that moves data between systems. When a form is submitted, n8n sends data to an AI agent, then posts the result to Slack.

Zapier

No-code automation connecting 6,000+ apps. Easy AI integrations and a great starting point for non-developers.

Airtable

A spreadsheet-database hybrid. AI workflows use it as a source of truth — pulling records in, running them through an agent, and writing results back out.

GitHub

Think Google Drive for code. Stores workflow code and agent logic, tracks changes over time, and enables team collaboration and rollbacks.

Notion

All-in-one workspace for notes, docs, and databases. AI agents can read from and write to Notion — great as a "second brain" for knowledge storage.

Anthropic API

Direct integration for developers building custom AI workflows. Use the SDK to call Claude in one line of code instead of writing raw HTTP requests.

Common workflow patterns

Trigger → AI → Action
New form submission → Claude summarizes and categorizes it → Creates a Notion page
Scheduled batch
Every morning → AI reads overnight emails → Produces a priority summary in Slack
Human in the loop
AI drafts a contract clause → Sends to a human for review → Human approves → Auto-sends to client
Example — Daily weather alert
Every day at 8:00 AM → n8n triggers automatically → Calls the OpenWeatherMap API to fetch current weather in Austin, TX → Formats the data → Posts a Slack notification to your chosen channel with the day's forecast. No manual steps required.

Getting started tips

Start with a single, repetitive task you do manually every week.
Map out the steps: what triggers it, what data is involved, what the output should be.
Build a simple version in n8n or Zapier with an AI step in the middle.
Test thoroughly, add error handling, and gradually hand off more to the automation.
